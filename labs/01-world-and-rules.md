# Лаб 01. Осмотр королевства

## Цель

Понять, какие части кластера уже существуют.

## Команды

```bash
kubectl cluster-info
kubectl get nodes -o wide
kubectl get namespaces
kubectl get pods -A
```

## Что посмотреть

- сколько узлов в кластере;
- где живут системные `Pod`;
- какие namespaces уже созданы.

## Что должно получиться

Вы видите, что Kubernetes уже состоит из многих системных компонентов, даже до запуска вашего приложения.
