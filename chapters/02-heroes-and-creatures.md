# 02. Герои и существа

## Сюжет

Совет магов разрешает вам призвать первого героя в королевстве. Он будет жить на одном из узлов и выполнять ровно одну роль.

## Что это в мире D&D

Один герой, один монстр или один NPC в лагере - это хороший образ для `Pod`.

## Что это в Kubernetes

`Pod` - минимальная сущность, которую Kubernetes размещает на узле. Обычно внутри `Pod` живет один основной контейнер, но Kubernetes управляет ими как одной единицей.

Важно:

- `Pod` эфемерен;
- его легко создать и заменить;
- напрямую управлять большим количеством одиночных `Pod` неудобно.

## Что вы делаете руками

- создаете первый `Pod` `quest-scroll`;
- смотрите, что контейнер запустился;
- получаете логи и описание.

## Минимальный YAML

См. [manifests/01-pod/quest-scroll-pod.yaml](../manifests/01-pod/quest-scroll-pod.yaml).

## Команды проверки

```bash
kubectl apply -f manifests/01-pod/quest-scroll-pod.yaml
kubectl get pods -n adventure
kubectl describe pod quest-scroll -n adventure
kubectl logs quest-scroll -n adventure
```

## Что должно получиться

- `Pod` в статусе `Running`;
- внутри работает `nginx`;
- по `kubectl logs` можно увидеть стандартный лог веб-сервера.
