# Словарь терминов

| D&D образ | Kubernetes термин | Что это значит |
| --- | --- | --- |
| Королевство | Cluster | Полная система Kubernetes |
| Совет магов | Control plane | Компоненты управления кластером |
| Остров или земля | Node | Узел, где реально запускаются контейнеры |
| Герой или существо | Pod | Минимальная единица запуска |
| Отряд | Deployment | Управляемая группа одинаковых Pod |
| Список бойцов | ReplicaSet | Механизм поддержки нужного числа Pod |
| Дорога или ворота | Service | Стабильная точка доступа к Pod |
| Свиток | ConfigMap | Открытая конфигурация |
| Запечатанное письмо | Secret | Чувствительные данные |
| Портал | Ingress | Внешний HTTP-маршрут до сервиса |
| Сокровищница | Volume / PVC | Подключенное хранилище |
