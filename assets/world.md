# Мир Кубернесии

Кубернесия устроена как архипелаг узлов. Между островами путешествуют отряды, но их маршруты не определяются вручную для каждого бойца. Вместо этого Совет Контрольного Плана следит за желаемым порядком мира и пересобирает лагери, когда кто-то исчезает.

Учебная модель мира:

- один локальный кластер `kind`;
- namespace `adventure` как основная игровая зона;
- приложение `quest-board` как главный артефакт кампании;
- ingress-controller как городские врата;
- том `/data` как сокровищница заметок партии.

Этот файл удобно использовать как краткое описание проекта, если позже репозиторий будет перенесен на docs-сайт.
